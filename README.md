# Tropical-Basin-Erosion-Modeling-Pipeline
This repository contains the GEE and Python framework for mapping gully susceptibility in Anambra, Nigeria. By integrating SRTM and SoilGrids data, the pipeline compares Random Forest and BLR models to identify erosion drivers in the Idemili and Awka regions. Features include automated covariate extraction and 3D-verified gully inventories.
